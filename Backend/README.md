# Node.js Backend

Production-level Node.js backend using Express and CommonJS.

## Setup

1. Install dependencies: `npm install`
2. Configure `.env` with your environment variables (refer to the existing `.env` file).
3. Run dev server: `npm run dev`
4. Start production server: `npm start`
